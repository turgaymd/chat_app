A full-stack chat application was built with Next.js, TypeScript, Node.js, and Socket.io.
Features:
- User authentication
- Real-time messaging
- Emoji, image and audio message support
- Online/Offline user tracking
- Message deletion
Tech stack:
 Frontend:
 - Next.js
 - TypeScript
 - Tailwind CSS
 Backend:
 - Node.js
 - Express.js
